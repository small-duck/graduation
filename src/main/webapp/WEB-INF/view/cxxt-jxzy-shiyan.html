Snort入侵检测软件

  Snort是个开源的IDS。它由4个部分组成：嗅探器、预处理器、检测引擎以及报警。嗅探器行为非常类似于ethereal或者tcpdump。它会将通信转储到屏幕或其它指定的位置。它用来聚集由预处理器和检测引擎所分析的通信。

NS2网络仿真软件

   NS2 是指 Network Simulator version 2，它是一种针对网络技术的源代码公开、免费的 软件仿真平台，研究人员使用它很容易进行网络技术研发，而且发展到今天，它所包含的模 块几乎涉及到网络技术的所有方面。NS2 是目前广泛使用的一种网络仿真软件。此外，NS2 也可作为一种辅助教学的工具，逐步广泛应用在网络技术教学领域。总之，目前在学术界和 教育界，有许多人正在或试图使用NS2。我们主要使用它的无线仿真部分。
